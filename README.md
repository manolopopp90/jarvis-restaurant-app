# Restaurant-App "Dä Seeblick" - Berg SG

## Projekt-Übersicht
Moderne Restaurant-App mit QR-Code Bestellung und Management-System.

## Features

### Gäste-App
- 📱 QR-Code Scan am Tisch
- 🍽️ Digitale Speise- & Getränkekarte
- 🛎️ Direktbestellung an den Tisch

### Wirt-App (Management)
- 📊 Grafische Tischplanung
- 🔄 Tisch-Status (Frei/Belegt/Offene Bestellung)
- 📝 Speisekarte verwalten
- 💰 Tagesumsatz-Anzeige
- 💾 SQLite Datenbank

## Technologie-Stack
- **Frontend:** React + Vite + Tailwind CSS
- **Backend:** Node.js + Express
- **Datenbank:** SQLite
- **QR-Code:** qrcode.js

## Projektstruktur
```
Restaurant-App/
├── frontend/          # React Gäste- & Wirt-App
├── backend/           # Node.js API
├── database/          # SQLite & Schema
└── docs/             # Dokumentation
```
